# Python API Projects

A collection of Python projects focused on working with APIs, HTTP requests, and JSON data.

These projects demonstrate backend Python skills including API integration, data handling, and error management.

---

## Projects

### Weather App
Fetches live weather data using the OpenWeather API.

### Stock Alert App
Retrieves real-time stock prices using the Alpha Vantage API.

### Crypto Price Tracker
Displays current Bitcoin price using a cryptocurrency API.

### News Fetcher
Fetches top news headlines from a news API.

### Simple REST API Practice
Practice project using GET and POST requests with a public API.

---

## Skills Used

- Python requests library
- REST APIs
- JSON handling
- HTTP requests
- Error handling

---

## Installation

git clone https://github.com/sarayadev/python-api-projects.git
cd python-api-projects
pip install requests

---

## How to Run

Each project can be run individually from its own folder.

Example:

cd weather-app
python main.py

Replace "weather-app" with the folder name of the project you want to run (e.g., stock-alert, crypto-tracker, news-fetcher).

If an API key is required:
- Create a .env or config.py file (depending on the project)
- Add your API key inside it
- Run the script again
